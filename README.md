# Ripgrep Replace - VS Code Extension

Ripgrep mantığında çalışan, kod dosyalarında arama ve değiştirme yapan VS Code eklentisi.

## Özellikler

- Tüm proje dosyalarını tarar
- Regex destekli arama
- Tek seferde tüm dosyalarda değişiklik yapar
- `.ts`, `.js`, `.tsx`, `.jsx`, `.py`, `.java`, `.cs`, `.cpp`, `.go` dosyalarını destekler
- `node_modules` ve `.git` klasörlerini otomatik atlar
